== Description ==

This plugin enables you to remind you old posts on the same day year(s) ago via email. Old  posts remind you that feeling is a little shy and nostalgic, bringing back you to old days.

== Installation ==

1. Upload `wp-remember-your-posts.php` to the `/wp-content/plugins/wp-remember-your-posts` directory 
1. Activate the plugin through the 'Plugins' menu of WordPress
